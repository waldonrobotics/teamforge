# FTC TeamForge v1.1.0

**Release Date:** November 13, 2024

## What's New

This release adds three major features:

1. **Version Tracking System** -
   From now on, when the app is updated, you'll see a dialog if database changes are needed. This makes updates smoother and safer.

2. **Intelligent Vendor Tracking** -
   When adding expenses, you can now track vendors. The system will autocomplete from previously used vendors, making data entry faster and more consistent.

3. **Download expenses as CSV** -
   You can now download expenses as CSV for a given time range

## Bug Fixes

This release fixes the following bug:

1. **Recurring events not working** - Recurring events were not working correctly earlier. That has now been fixed.

**NOTE** - This release requires database changes. Please run the migration script from the tab above in Supabase to continue.
