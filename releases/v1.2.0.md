# FTC TeamForge v1.2.0

**Release Date:** November 14, 2024

## What's New

This release adds one major feature:

1. **Scouting System** -
   Scout other FTC teams at competitions with customizable templates. Create templates with different question types (text, numbers, multiple choice, images, field diagrams), fill out sheets for teams you're competing against, and view all your scouting data organized by team and event.

**NOTE** - This release requires database changes. Please run the migration script from the tab above in Supabase to continue.
