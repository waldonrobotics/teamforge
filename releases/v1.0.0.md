# FTC TeamForge v1.0.0 - Initial Release

**Release Date:** November 13, 2024

## Overview

Welcome to the first official release of FTC TeamForge! This is the baseline version that includes all core features for managing your FTC robotics team.

## Features Included

### Core Team Management
- Team creation and configuration
- Team member management with roles (Admin, Mentor, Student, Guest)
- Team invite system with custom invite codes
- Multi-season support

### Team Notebook
- Rich text editor with BlockNote (Notion-like interface)
- Folder-based organization with color coding
- Entity-linked notebook pages (tasks, events, mentoring sessions)
- Auto-save functionality
- Search across all notes

### Calendar & Events
- Full calendar view (month, week, day, agenda)
- Event types: meetings, competitions, outreach, workshops, social
- RSVP system and attendee tracking
- Recurring events support

### Task Management
- Kanban-style board (To Do, In Progress, Done)
- Task categories and assignments
- Due dates and tracking
- Linked notebook pages

### Budget & Fundraising
- Expense tracking with categories
- Fundraising opportunity management
- Budget utilization charts
- Financial reporting

### Mentoring Module
- Mentored team management
- Mentoring session tracking
- Hours logging and analytics

### Scouting System
- FTC Events API integration
- Team search and information lookup
- Event search and team rosters
- Match history and awards tracking

### Dashboard
- Team metrics and KPIs
- Financial overview
- Task breakdown charts
- Upcoming events and tasks

### User Settings
- Profile management
- Theme customization (light/dark mode)
- Accent color selection
- Password management

## Technical Details

- Built with Next.js 15, React 19, and TypeScript
- Supabase backend with PostgreSQL
- Row Level Security (RLS) for data isolation
- Offline-first architecture with React Query
- Mobile-responsive design

## Database Schema

This version establishes the baseline database schema including:
- 16 core tables
- 81 Row Level Security policies
- 9 database functions
- Complete indexes and constraints

## Installation

For new installations, the First Run Experience (FRE) will guide you through:
1. Database setup
2. Team creation
3. Season configuration
4. Admin user setup

## Notes

This is the baseline version. All existing installations are considered to be running v1.0.0.
